# Regional Property Intelligence Platform (RPIP)

## Региональная платформа пространственной аналитики недвижимости

---

## О проекте

Regional Property Intelligence Platform (RPIP) — открытая модульная платформа пространственного анализа рынка недвижимости, создаваемая государственным учреждением для поддержки государственной кадастровой оценки объектов недвижимости.

Проект ориентирован на создание единого цифрового пространства данных о недвижимости региона.

Основная цель платформы — объединение пространственных, кадастровых, рыночных и аналитических данных в единой информационной системе с использованием свободного программного обеспечения.

---

## Основные задачи

Платформа обеспечивает:

- ведение единого регионального озера данных;
- хранение пространственных данных;
- накопление истории изменений объектов;
- анализ рынка недвижимости;
- расчет пространственных факторов стоимости;
- поддержку государственной кадастровой оценки;
- контроль качества данных;
- пространственную аналитику;
- подготовку аналитических отчетов.

---

## Основные принципы

Проект строится на следующих принципах.

### Data First

Главной ценностью являются данные.

Интерфейс может изменяться.

Данные остаются.

---

### Open Source First

Используется свободное программное обеспечение.

Предпочтительные технологии:

- PostgreSQL
- PostGIS
- Python
- QGIS
- GeoServer
- FastAPI
- Docker

---

### Spatial First

Все данные рассматриваются как пространственные.

Любой объект имеет геометрию или пространственную привязку.

---

### Explainable Analytics

Любой аналитический результат должен быть воспроизводим.

---

### Incremental Development

Каждая версия платформы должна использоваться в ежедневной работе учреждения.

---

## MVP

Первая версия платформы реализует следующие возможности.

- импорт данных ЕГРН;
- хранение пространственных объектов;
- цифровой паспорт объекта недвижимости;
- отображение объекта на карте;
- поиск по кадастровому номеру;
- импорт рыночных наблюдений;
- расчет базовых пространственных факторов.
